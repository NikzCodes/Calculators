## AMS Retainer Calculator

Calculates agency retainer pricing for Amazon Marketing Services with:
- **Base retainer** (configurable, default ₹10K)
- **Variable fee** on incremental sales (total sales minus existing organic sales)
- **TACoS-based tiers** — lower TACoS = higher variable %, rewarding efficient ad spend

### Formula
```
Retainer = Base + (Incremental Sales × Tier%)
Incremental Sales = Total Sales − Organic Sales
```

### TACoS Tiers
| TACoS Range | Variable Fee % |
|---|---|
| Under 15% | 3.0% |
| 15% – 20% | 2.0% |
| 20% – 25% | 1.5% |
| 25% – 30% | 1.0% |
| Over 30% | 0.5% |
